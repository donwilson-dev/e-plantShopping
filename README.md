e-plantShopping 🌱
📌 Project Overview

e-plantShopping is a web-based application that allows users to browse and select a variety of plants for purchase. The app provides an interactive shopping experience where users can view plant details, add items to a cart, and manage their selections efficiently.

This project was developed as part of the Coursera learning program to demonstrate skills in front-end development and application state management.

🚀 Features
Browse a collection of plants
View plant details (name, price, etc.)
Add plants to a shopping cart
Update item quantities in the cart
Dynamic total cost calculation
🛠️ Technologies Used
React (JavaScript library for building UI)
Redux (state management)
HTML5 & CSS3
Vite (build tool)
📂 Repository

This project is stored in the GitHub repository: e-plantShopping

▶️ How to Run the Project

Clone the repository:

git clone https://github.com/YOUR_USERNAME/e-plantShopping.git

Navigate to the project folder:

cd e-plantShopping

Install dependencies:

npm install

Run the development server:

npm run dev
